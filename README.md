# slack-mahjong-score-management

## 概要

slackに投稿されたスコアを集計するツール

「御無礼成績」で今月の成績（トータルポイント集計）  
「御無礼記録」「御無礼結果」でスプレッドシート貼り付け用集計済みデータ  
「御無礼グラフ」でその日のポイント推移グラフ

をチャンネル内に表示する。
